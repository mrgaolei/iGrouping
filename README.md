iGrouping
=========
An iOS project that can grouping many peoples to each group.
Support config how many groups you are grouping and number of each group.


还在为分拨分组烦恼吗？还在用老掉牙的cei丁壳和手心手背？还在用6年的硬笔书法功底裁纸做抓阄签？
别out了，用本app可以解决你一切烦恼。
支持定义每组多少人和共有多少组。
